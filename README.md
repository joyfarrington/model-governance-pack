# Model Governance Pack

A practical starter kit of AI model governance templates, plus a completed example for a fictional system.

## What’s inside
- `/templates` — reusable governance templates (model card, intended use, etc.)
- `/examples` — a filled-out example showing how the docs look when completed

## Who this is for
- Anyone learning AI governance / GRC
- Teams that want a lightweight baseline for documentation and review

## How to use
1. Copy what you need from `/templates`
2. Fill it out during design and before launch
3. Use `/examples` as the reference for tone and detail

## Disclaimer
This repo is educational and operational guidance, not legal advice.
