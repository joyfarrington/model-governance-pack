# Intended Use

## Allowed use cases
- Answer FAQs using the help-center as the source of truth
- Help users navigate scheduling, billing, and account questions
- Summarize a user’s issue for a human agent during handoff
- Route requests to the right support queue

## Disallowed use cases
- Medical, legal, or financial advice
- Collecting or requesting sensitive personal data (SSN, full medical details, etc.)
- Making eligibility, coverage, or benefits decisions
- Persuasion or manipulation (pressure tactics, emotional coercion)

## Out-of-scope
- Diagnosis, treatment guidance, or symptom triage
- Any decision-making that impacts access to care or coverage
- Anything requiring professional judgment

## Users and context
- **Primary users:** Customers and support agents
- **Where it will be used:** Customer-facing chat widget with “talk to a human” option

## User transparency
Users see a clear notice that this is an AI assistant, it can be wrong, and they can request a human at any time.
